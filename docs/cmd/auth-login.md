# safedep auth login

Authenticate with SafeDep Cloud and store credentials in the keychain under the active profile.

> Status: not yet implemented. The command is wired into the CLI tree but returns an error on use.

## Synopsis

```
safedep auth login [--profile <name>]
```

## Flags

Inherits root flags `--output` and `--profile`.

## Exit codes

- `0` on success.
- `1` on any failure.
