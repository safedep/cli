# safedep auth profile list

List the credential profiles known to this CLI.

> Status: not yet implemented. The command is wired into the CLI tree but returns an error on use.

## Synopsis

```
safedep auth profile list
```

## Flags

Inherits root flags `--output`, `--profile`, and `--insecure-keychain-fallback`.

## Exit codes

- `0` on success.
- `1` on any failure.
