# safedep subscription ondemand disable

## Synopsis

```
safedep subscription ondemand disable
```

## Description

`subscription ondemand disable` opts the tenant account out of usage-based overage
billing. Included seat limits continue to apply.

## Examples

```
safedep subscription ondemand disable
```

## Exit codes

| Code | Meaning |
|------|---------|
| `0` | On-demand billing disabled. |
| non-zero | RPC error. |
